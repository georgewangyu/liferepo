# Shareable Writing

## Genesis

Created after the need for a clean place to draft markdown pieces about the
`liferepo` / agent-system philosophy before publishing them as gists or other
shareable posts.

Inspiration: "can we write that into a gist somewhere ... maybe we start a
folder for markdown files that we can share"

## Purpose

This directory holds public-safe markdown drafts that are meant to leave the
repo eventually.

Typical uses:

- gist drafts
- public notes on agent design
- short essays about context, memory, and workflow design

## Structure

- `AGENT-shareable.md` defines writing rules for this area
- `IMPROVEMENTS.md` tracks backlog ideas for stronger publish workflows
- individual markdown files hold one shareable piece each
