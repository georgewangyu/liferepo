# Long-Form Writing Workflow

Reusable workflow for long-form writing (blog/newsletter/articles).

## Target Outcome

Create a high-signal long-form piece with strong structure, defensible claims,
and a clear reader takeaway.

## Phase 0: Planning Inputs

1. Audience profile and pain points.
2. Primary thesis.
3. Desired CTA.
4. Evidence/examples available.

## Phase 1: Structure Design

Build structure before prose:

1. Hook.
2. Beat 1 (setup).
3. Beat 2 (development).
4. Beat 3 (problem/gap tension).
5. Twist (reframe).
6. Resolution (implication + CTA).

Do not draft full prose until structure is stable.

## Phase 2: Drafting

1. Write section-by-section.
2. Keep each section focused on one point.
3. Include specific examples and mechanism-level explanations.
4. Maintain consistent narrative progression.

## Phase 3: Review

1. Remove repeated ideas and filler.
2. Validate factual claims.
3. Improve transitions between sections.
4. Tighten opening and ending for clarity and momentum.

## Reusable Prompt Frames

- 4C planning prompt:
  - context
  - command
  - constraints
  - creativity
- Section expansion prompt:
  - main point
  - supporting evidence
  - transition
  - reader value
- Fact-check pass prompt:
  - claim inventory
  - verification status
  - rewrite suggestions

## Output Contract

Deliver:

- final article draft
- top edits made in revision pass
- unresolved assumptions or evidence gaps
