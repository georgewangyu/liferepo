# Platform Promotion Workflow

Reusable workflow for promoting long-form content across X, LinkedIn, and
Threads without leaking private profile details.

## Objective

Turn one long-form source asset into multiple short-form promotion assets that
drive qualified clicks and subscriptions.

## Input Contract

Before promotion drafting, collect:

1. Source asset URL/slug and publish status.
2. Three to five key insights/snippets from the source.
3. Target audience and desired CTA.

## Snippet Selection Rules

Good snippets:

- stand alone without full-article context
- contain a concrete insight, mechanism, or story moment
- create curiosity without giving away the full conclusion

Reject snippets that are generic, context-fragile, or already reveal the full
article thesis.

## Platform Formatting

### X

- Start with a strong hook in compact format.
- Use continuation replies only when needed for clarity.
- Put the destination link in a follow-up reply if link suppression is a risk.

### LinkedIn

- Use a professional but direct tone.
- Prefer short paragraphs with intentional line breaks.
- End with one discussion prompt or clear CTA.

### Threads

- Use concise narrative progression similar to X.
- Keep pacing fast and conversational.
- Preserve one clear takeaway even in thread format.

## Drafting Workflow

1. Generate 3 hooks per platform.
2. Select one hook and draft full platform variant.
3. Validate CTA placement and link strategy.
4. Produce final posting set (X, LinkedIn, Threads).
5. Track conversions by format and hook type.

## Output Contract

For each source asset, produce:

- one X version
- one LinkedIn version
- one Threads version
- one sentence rationale for each variant
