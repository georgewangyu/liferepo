# Agents Directory

Canonical home for reusable agent-area definitions and taxonomy docs.

## Purpose

- Keep public-safe agent behavior in `liferepo`.
- Keep private/person-specific overlays in `<private-repo>`.
- Keep executable automation in `georgeskills`.

## Catalog

- `AGENT-agents.md`: Governance for this catalog.
- `MIGRATION_MAP.md`: Legacy `<private-repo>` agent path to canonical mapping.
- `business/`: Venture planning and execution workflows.
- `career/`: Resume, job-hunting, and interview prep workflows.
- `communications/`: Email and calendar workflow agents.
- `data-ownership/`: Data export/migration and privacy workflows.
- `housing/`: Property-document and housing operations workflows.
- `knowledge/`: Research and knowledge-ops workflows.
- `principles/`: Principles authoring and maintenance workflows.
- `social-media/`: Text/video distribution workflows.
- `writing/`: Cross-format writing quality and long-form workflows.

## Pattern

Suggested pattern for each area:
- `AGENT-<area>.md` for behavior and workflow rules.
- `README.md` for structure and usage.
- `IMPROVEMENTS.md` for backlog.
