# Social Media Agent Spec

Reusable guidance for cross-platform content workflows.

- Canonical agent rules: `AGENT-social-media.md`
- Platform promotion workflow: `PLATFORM_PROMOTION_WORKFLOW.md`
- Long-form distribution workflow: `SUBSTACK_DISTRIBUTION_WORKFLOW.md`

## Scope

- Short-form video story design
- Short-form text promotion across X/LinkedIn/Threads
- Long-form article/newsletter planning and launch sequencing

## Boundary

- Public reusable process rules live here.
- Private creator profile details and account performance data stay in
  `<private-repo>/social-media/`.
