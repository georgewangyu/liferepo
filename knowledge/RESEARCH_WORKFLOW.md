# Research Workflow

Reusable workflow for topic-based research and refinement.

## Phase 1: Question Framing

1. Define research question and scope.
2. Capture assumptions and target decision impact.
3. Draft or select prompt templates.

## Phase 2: Source Collection

1. Gather sources and notes.
2. Track source provenance.
3. Separate facts, hypotheses, and open questions.

## Phase 3: Synthesis

1. Group findings into patterns/models.
2. Identify contradictions and confidence levels.
3. Produce concise structured summary.

## Phase 4: Routing

Route outputs based on type:

- decision heuristics -> principles
- execution changes -> projects/workflows
- personal reflection -> journal
- reusable model -> knowledge framework docs

## Quality Rules

- Keep exploratory drafts but clearly mark maturity level.
- Preserve evidence links for factual claims.
- Prefer retrieval-friendly formatting over long narrative dumps.
